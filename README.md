## 📋 Overview

NEONO is a premium, agency-grade single-page restaurant website that showcases a futuristic luxury dining brand. Built entirely with vanilla HTML5, CSS3, and JavaScript - no frameworks or libraries - this project demonstrates modern web development techniques with a focus on performance, accessibility, and stunning visual design.

### ✨ Key Features

- **🎨 Modern Design Aesthetic**
  - Dark theme with gold/neon accents
  - Glassmorphism effects
  - Custom cursor for desktop
  - Premium typography (Cormorant Garamond + Inter)

- **📱 Fully Responsive**
  - Mobile-first approach
  - Hamburger menu for mobile devices
  - Fluid typography and spacing
  - Optimized for all screen sizes

- **⚡ Interactive Elements**
  - Dynamic menu with category filtering
  - Interactive gallery with lightbox
  - Testimonial slider with navigation
  - Reservation form with validation
  - Newsletter subscription

- **✨ Animations & Effects**
  - Scroll reveal animations
  - Parallax hero background
  - Hover glow effects
  - Smooth page transitions
  - Custom cursor interactions

- **🔍 SEO Optimized**
  - Comprehensive meta tags
  - JSON-LD structured data
  - Open Graph and Twitter cards
  - Semantic HTML5 structure

- **♿ Accessibility**
  - ARIA labels
  - Proper heading hierarchy
  - High contrast ratios
  - Keyboard navigation support
